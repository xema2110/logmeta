If you like Goldentowns and want a piece of it, you can buy game shares. Every time a player spends gold buying an advantage, the gold goes into a fund called the game fund. When this fund reach 100 gold, it will pay dividends to shareholders. There are 1.000.000 game shares, of which 500.000 can be traded. In this page, you can buy / sell game fund shares and also check the funds income.
