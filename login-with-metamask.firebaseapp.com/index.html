If you like Crypto Nations and want a piece of it, you can buy game shares. Every time a player spends ETH buying an advantage, the ETH goes into a fund called the game fund. When this fund reach 1 ETH, it will pay dividends to shareholders. There are 1.000.000 game shares, of which 750.000 can be traded. In this page, you can buy / sell game fund shares and also check the funds income.
